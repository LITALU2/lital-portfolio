# lital-portfolio

This repository contains my personal portfolio website, designed and developed for the "Vibe Coding" course at Shenkar College (M.Des in Digital Game Design and Development).

**Built With:**
* Semantic HTML5
* CSS3 (Flexbox, CSS Grid)
* Vanilla JavaScript (DOM manipulation, Custom Cursor, Modals)
* Hosted on GitHub Pages

**How to Run the Project:**
Click the link below to view the deployed website:
https://litalu2.github.io/lital-portfolio/
or
Clone this repository or download the ZIP file.
Extract the files to a local folder.
Open the index.html file directly in any modern web browser.
No build processes or local servers are required!
